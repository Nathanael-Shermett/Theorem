TransposableTrait

TransposableTrait.php : 11

Provides functionality for allowing {@see Note} entities to be transposed.

NOTE: TransposableTrait only provides functionality for transposing individual notes. If you need to transpose collections of notes (such as \Theorem\Chord and \Theorem\Sequence), use \Theorem\TransposableColletionTrait.
