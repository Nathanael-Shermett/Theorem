RenderableTrait

RenderableTrait.php : 12

Provides functionality for allowing entities (such as {@see Note} and {@see Chord}) to be rendered (i.e. printed to a string).

NOTE: RenderableTrait does not provide a default implementation of toString(). Any classes using RenderableTrait must write their own implementation.
