getSpn()

Note.php : 225

Returns the note's value in scientific pitch notation (SPN).

public getSpn() : string

NOTE #1: In SPN, octave numbers increment at C, not A.

NOTE #2: In SPN, accidental adjustments apply after determining the note's octave. Therefore, because octave numbers increment at C and accidentals apply afterwards, B4 and Cb5 are exactly one octave apart.
